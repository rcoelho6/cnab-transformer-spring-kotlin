# SANTANDER OPENWORLD — ÍNDICE

Documentação focada em integrações assíncronas (Queue / REST) para pagamentos usando APIs Santander (BaaS / Developer Portal / Open Finance).

APIs cobertas

- CORBAN DIGITAL (cobrança/conciliação)
- PIX TRANSFERS (pagamentos via PIX)
- TED TRANSFERS (TED)
- CONTAS E TRIBUTOS (contas, tributos, cobranças fiscais)
- DDA (Débito Direto Autorizado)
- TRANSF INTELIGENTES (fluxos avançados de transferência)

- `BAAS_PORTAL_OPEN.md` — resumo da interação BaaS / Developer Portal / Open Finance
